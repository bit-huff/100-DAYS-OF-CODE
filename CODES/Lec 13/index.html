Miscellaneous Tags

HTML Code Tag

The <code> tag in HTML is used to define a piece of computer code. It represents inline code snippets and is typically styled in a monoscaped (fixed-width) font to distinguish it from the rest of the text.

Syntax:-

<code>This is some code!</code>

Example:-

  • To print "Hello World!" in python, use the following code:

<code>print("Hello World!")</code>

HTML Entities

They enable you to display characters that are reserved in HTML or that aren't readily available on the keyboard.

What are HTML entities?

HTML Entities are used to represent special characters in a format that the browser can understand.
They start with an amperstand(&) and end with semicolon(;).

Why use HTML Entities?

  • Reserved Characters: Characters like <, >, and & are reserved in HTML.
  • Special Symbols: For Symbols like © or any mathematical symbols.
  • Non-Breaking Lines: To create while spaes that won't break into a new line.

Quotation Tags

The use of quotations is common in textual content.
HTML provides specifie tags to handle this:

  • <blockquote>: For block quotations
  • <q>: For inline quotations

Example with syntax:-

1. blockquote Tag:-

Fake it till you make it.
=>Syntax code:

<blockquote> Fake it till you make it. </blockquote>

2. q Tag:-

          I'm faster than you think.

=>Syntax code:

<q>I'm faster than you think.</q>
