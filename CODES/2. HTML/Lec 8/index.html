Block Elements

Block elements are those that typically start on a new line and take up the full width
available. They create a "block" of content and stack on top of each other vertically.
They are used to structure the main layout of the page. Common block elements include:

  • div: A generic container for grouping content.
  • p: Defines a paragraph.
  • h1, h2, h3, h4, h5, h6: Define headings of different levels.
  • ul, ol: Define unordered and ordered lists, respectively.
  • li: Defines a list item within a list.
  • header, footer, main, section, article, aside: Semantic elements for different sections of a page.

Inline Elements

Inline elements, on the other hand, do not start on a new line. They only take up as much
width as necessary and flow along with the surrounding content. Inline elements are often
used to style or format parts of text within a block element. Examples include:

  • span: A generic inline container for styling or grouping small sections of text.
  • a: Defines a hyperlink.
  • strong: Makes text bold and indicates strong importance.
  • em: Italicizes text and indicates emphasis.
  • img: Embeds an image in the document.
  • br: Inserts a line break within text.
