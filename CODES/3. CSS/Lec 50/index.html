CSS Filters

CSS filters are graphical effects applied to elements such as images, backgrounds, and videos. They allow you to manipulate elements by changing their appearance in ways like adjusting color, blurring, or enhancing contrast.

Here are the most commonly used CSS filter functions:

Blur
Brightness
Contrast
Grayscale
Hue-rotate
Sepia
Invert
Opacity
Saturate
Drop-shadow
