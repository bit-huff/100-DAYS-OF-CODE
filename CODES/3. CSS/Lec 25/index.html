You can customize the appearance of lists using various CSS Properties. Lists can be ordered of <ol> or unnordered <ul> , and with CSS, you can control how they look.

  1. List style Type:

    2. The list type property defines the type of marker (Bullet or Number) for the list items. 

    ul                                          ol
    {                                           {
        list-style-type:circle;                  list-style-type:decimal;
    }                                           }
  2. Custom List Marker:

    3. Use images or cutsom icons as list markers by using list-style-image!

    ol
    {
        list-style-image:URL('PATH/TO/YOUR-IMAGE.png')
    }
  3. Removing List Marker

    4. To remove default bullets or numbers.

    ul, ol 
        {
            list-style-type:none;
        }
  4. Horizontal Lists

    5. By default, list markers inhertit the text color. If you want to style the marker color independently use the ::marker pseudo element.

    ul li::marker
                {
                    color:red;
                }
