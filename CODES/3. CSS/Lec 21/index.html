CSS Specificity

There is a list based on Specificity of selectors!

  • Inline Stles: 1000 points
  • ID Selectors: 100 points
  • Class/Attributes/Pseudo Class Selectors: 10 points
    (In case of two or more than two classes algorithm apply by considering position and order of appearance!)
  • Element Selectors:1 points
  • Universal Selectors:0 points

Note: !important rule has the highest Specificity (10,000 points)

p { color:red; !important }
p { color:blue; }
