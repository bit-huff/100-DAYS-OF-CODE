Overflow Property

The CSS overflow property controls what happens to content that overlows the bound of an element. It applies to block-level elements like dev, section or any other element with a defined size.

There are four main values for the overflow property:-

1.Visible (Default)

The content is not clipped and can overflow the elements box. The overflowing content will be visible outside of the element. 

        div {
            overflow:visible;
        }

2. Hidden

The content is clipped, it means any content that overflow the elements box will be hidden and not visible. 

            div {
                overflow:hidden;
            }

3. Scroll

The content is clipped, but a scrollbar is added to allow users to scroll through the content even if it overflows. 

            div {
                overflow:scroll;
            }

4. Auto

IF the content overflows the elements box, a scrollbar is automatically added. If the content fits within the element, no scrollbar is appears. 

            div {
                overflow:auto;
            }

Additionally, you can control overflow on the horizontal and vertical axex seperately using overflow-x and overflow-y.

  • Overflow-x: Control the overflow in the horizontal direction (left to right)
  • overflow-y: Control the overflow in the vertical direction (top to bottom)
For eg:            div  {
                         overflow-x:hidden;
                         overflow-y:auto;
                        }

It hides horizontal overflow but allow vertical scrolling if the content exceeds the container's height.
