CSS provides a variety of units to control the sizing of elements, each suited for different contents like fixed, realative, or responsive design.

There are genarally 9 types of sizing units which used mostly.

  1. px: Fixed sized in pixels
  2. rem: Relative to root font size
  3. em: Relative to parent element font size
  4. min-height: Set the minimum height element can have
  5. min-width: Set the minimum width element can have
  6. vw: 1% of the viewport width
  7. vh: 1% of the viewport height
  8. vmin: 1% of smaller dimension of viewport
  9. vmax: 1% of larger dimension of viewport
